Cozy Comfort ERP
----------------

CozyComfort_ERP is an ERP system designed for Sellers, Manufacturers, and Distributors to manage orders, inventory, and blanket requests efficiently. It uses C# and ASP.NET Web Forms with SOAP-based web services and SQL Server as the database.

🛠️ Tech Stack
--------------

- **Frontend & Backend:** C#, ASP.NET Web Forms
- **Web Services:** SOAP (ASMX)
- **Database:** Microsoft SQL Server
- **IDE:** Visual Studio

👤 User Roles & Features
-------------------------
### Seller
- Add new orders
- View current stock
- Manage inventory manually
- Track pending and confirmed orders
- Request blankets
- Receive stock updates from distributors

### Distributor
- View and confirm seller orders
- Track current stock
- Manage inventory
- Handle rejected or pending seller requests

### Manufacturer
- View and confirm distributor orders
- Track current stock
- Manage inventory
- Handle rejected requests from distributors

🔗 Notes
---------
Make sure the web services are running before logging in.
